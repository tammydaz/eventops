# Menu reference (Claude / internal)

## Why `Pasted text.txt` might not open

`Pasted text.txt` is **one very long line** (~60k characters). Some editors (including Cursor/VS Code in certain modes) **struggle or appear blank** with extremely long lines.

## What to use instead

Open **`foodwerx-menu-readable.txt`** — same content, **word-wrapped** to short lines so it opens and scrolls normally.

The original `Pasted text.txt` is kept unchanged if you need a byte-for-byte export.
