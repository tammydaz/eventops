# AI Bootstrap

Before giving answers, AI assistants should load the following files in order:

1. **SYSTEM_CONTEXT.md** — High-level overview of the EventOps system architecture, core tables, stack, and workflows.
2. **BUILD_STATE.md** — Current development phase, completed features, work in progress, and next steps.
3. **RULES.md** — Strict architecture rules and constraints for the EventOps system.

Read these files at the start of each session to ensure context-aware and rule-compliant responses.
